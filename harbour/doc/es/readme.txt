 *                        <b>El Proyecto Harbour</b>   </par>
 *
 * <b>**********************************************************************</b>   </par>
 * <b>*  Este archivo contiene información sobre como obtener, instalar y  *</b>   </par>
 * <b>*  usar Harbour. Por favor léalo *completamente* antes de solicitar  *</b>   </par>
 * <b>*  ayuda                                                             *</b>   </par>
 * <b>**********************************************************************</b>   </par>
 *
 * Harbour es una implementación libre (en todo sentido) de un compilador
 * de lenguaje xBase. Este está designado para ser compatible con el
 * código fuente del compilador CA-Clipper(c).
 * Esto significa que si Ud. tiene algún código que debería compilar usando
 * CA-Clipper(c) entonces éste debería compilar también bajo Harbour.
 *
 * La página web del Proyecto Harbour es:   </par>
 *              <b>http://www.Harbour-Project.org/</b>   </par>
 *
 * El estado y cualquier otra información está siempre disponible desde
 * el sitio web.
 * Hay una lista de correo de Harbour. Harbour todavía está en una etapa
 * temprana de desarrollo, así que la lista de correo de los desarrolladores
 * es practicamente la unica lista, aunque cualquiera es bienvenido para
 * unirse a las discusiones.   </par>
 *
 * Nos gustaría que Ud. se uniera al equipo de desarrollo de Harbour.
 * Si Ud. esta interesado, puede subscribirse a nuestra lista de correo y
 * comenzar contribuyendo a este proyecto público libre.   </par>
 *
 * Por favor sientáse con libertad de hacer preguntas, reportar ideas,
 * sugestiones, correcciones, código, etc.
 * Con la ayuda de todos Uds., el compilador Harbour y las librerías de
 * ejecución se hará realidad muy pronto.   </par>
 *
 *
 * <b>Qué es lo que esta distribución contiene ?</b>   </par>
 * ==========================================   </par>
 *
 * Esta distribución es solamente una distribución de código fuente. Esta
 * no contiene ningún archivo ejecutable. Las versiones ejecutables de
 * Harbur estan dispobibles desde el sitio web.
 * Las versiones ejecutables de Harbour NO crean programas ejecutables
 * Harbour hasta este momento produce código de lenguaje C de salida,
 * el cual debe ser compilado con la VM (Máquina Virtual de Harbour) y
 * las librerías de soporte para crear programas funcionales.   </par>
 * Por favor haga tests, corriendo Harbour contra su código fuente en
 * Clipper y reportando cualquier problema que pudiera ocurrir.    </par>
 *
 * Muy importante: El pre-procesador esta ahora trabajando.
 *
 *
 * <b>Instalación</b>   </par>
 * ------------   </par>
 *
 * 1. Deszipee el archivo zip de Harbour usando pkunzip ó equivalente.  </par>
 *         Por ej.: pkunzip -d build37.zip   </par>
 *    Esto creará el directorio Harbour y todos los subdirectorios
 *    relevantes.   </par>
 *
 * 2. Compile Harbour usando su compilador de lenguaje C. Archivos make
 *    para diferentes plataformas están incluidos.   </par>
