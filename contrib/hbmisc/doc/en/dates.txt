   $CATEGORY$
      Date
   $ONELINER$
      Returns an array with the months names.
   $SYNTAX$
      AMonths() --> aMonths
   $ARGUMENTS$
      None
   $RETURNS$
      <aMonths> The array which holds the months names.
   $DESCRIPTION$
      This function returns an array with all the months names in the

   $EXAMPLES$

   $STATUS$

   $COMPLIANCE$
      This function is new in Harbour.

      All
   $FILES$
      Library is hbmisc
   $SEEALSO$
      ADays()
   $END$
